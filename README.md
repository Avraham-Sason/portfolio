# Avraham Sason — Portfolio

Personal portfolio website for Avraham Sason, Senior Full Stack Developer based in Elad, Israel.

**[→ Live Site](https://avrahahm-sason-portfolio.vercel.app/)**

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **UI Components:** shadcn/ui (Radix UI primitives)
- **Animations:** Framer Motion, GSAP
- **Analytics:** Vercel Analytics
- **Fonts:** Inter, JetBrains Mono

## Features

- Responsive, mobile-first design
- Dark / light theme support
- Internationalization (i18n) via `LanguageProvider`
- Animated hero section with a live code block
- Sections: Hero, About, Skills, Experience, Projects, Infrastructure, Contact, Footer
- OpenGraph & Twitter card metadata
- PWA manifest

## Project Structure

```
app/               # Next.js App Router pages & layout
components/
  portfolio/       # Portfolio section components
  ui/              # Reusable UI primitives (shadcn/ui)
lib/               # Utilities and i18n provider
public/            # Static assets
```


## Deployment

Deployed on [Vercel](https://vercel.com): [https://avrahahm-sason-portfolio.vercel.app/](https://avrahahm-sason-portfolio.vercel.app/)
